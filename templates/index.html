Our Project

Our Mission

The goal of this project is to revolutionize current catastrophe prediction techniques with machine learning. This web app provides functionality for wildfires, hurricanes, and earthquakes. A variety of machine learning models are used ranging from linear regression to random forest to convolutional neural networks.
